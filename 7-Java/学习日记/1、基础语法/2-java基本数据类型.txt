基本数据类型
	
	变量
		申请内存来存储值。创建变量的时候，需要在内存中申请空间

	内置数据类型（8种）
		btye 8位，主要用于整数
		short 16位，
		int 32位

		long 64位  默认值位0L
			
		float 单精度，32位，0.0f

		double 双精度，64位，0.0d

		boolean trur/false 默认false

		char 单一的16位Unicode字符
		
	引用数据类型
		类似指针
		
		对象，数组都是引用数据类型
		默认位null

	常量：使用
		final 来修饰


运算时，会发生自动类型转换
	
转换规则
	低-------------->>>>----------高

byte,short,char-> int->long->float->double


注意：
	不能对boolean类型进行类型转换
	不能把对象类型转换成不相关类的对象
	
	容量大的转容量小的必须使用强制类型转换
	
	转换容易发生丢失或者溢出精度

	浮点数变整数通过丢小数
