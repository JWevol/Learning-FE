java基本语法
	大小写敏感
	类名。首字母大写
	方法名。首字母小写，驼峰
	源文件名必须和类名相同。
	主方法入口 public static void main(String []args)

	大小写字母，美元符，下划线

	访问控制类型：	default public protected private
	非访问控制修饰符：final abstract strictfp

	局部
	类变量（静态）
	成员变量（非静态）
